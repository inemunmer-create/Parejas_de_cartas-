# Parejas_de_cartas-

# 	**DESCRIPCION DEL JUEGO**
 
Este proyecto consiste en un juego de memoria de parejas de cartas. El objetivo del juego es encontrar las cartas que tienen el mismo número, levantando dos cartas en cada turno hasta formar todas las parejas posibles.

#	**FUNCIONAMIENTO DEL JUEGO**
  
-Las cartas se colocan boca abajo.

-En cada turno, el jugador selecciona dos cartas.

-Si los números coinciden, se forma una pareja y las cartas permanecen visibles.

-Si no coinciden, las cartas vuelven a ocultarse.

-El juego termina cuando se han encontrado todas las parejas.
